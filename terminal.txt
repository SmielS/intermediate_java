1)   Используйте команды операционной системы Linux и создайте две новых директории – 
	 «Игрушки для школьников» и «Игрушки для дошколят»

mkdir 'Игрушки для школьников'
mkdir 'Игрушки для дошколят'

2)   Создайте в директории «Игрушки для школьников» текстовые файлы - «Роботы», «Конструктор», 
	 «Настольные игры»
	 
touch 'Роботы'.txt 'Конструктор'.txt 'Настольные игры'.txt

┌──(kali㉿kali)-[~/lws/attestation2/Игрушки для школьников]
└─$ ll
total 0
-rw-r--r-- 1 kali kali 0 Feb 27 08:03  Конструктор.txt
-rw-r--r-- 1 kali kali 0 Feb 27 08:03 'Настольные игры.txt'
-rw-r--r-- 1 kali kali 0 Feb 27 08:03  Роботы.txt

3)	 Создайте в директории «Игрушки для дошколят» текстовые файлы «Мягкие игрушки», «Куклы», «Машинки»

touch 'Мягкие игрушки'.txt 'Куклы'.txt 'Машинки'.txt

┌──(kali㉿kali)-[~/lws/attestation2/Игрушки для дошколят]
└─$ ll
total 0
-rw-r--r-- 1 kali kali 0 Feb 27 08:06  Куклы.txt
-rw-r--r-- 1 kali kali 0 Feb 27 08:06  Машинки.txt
-rw-r--r-- 1 kali kali 0 Feb 27 08:06 'Мягкие игрушки.txt'

4)   Объединить 2 директории в одну «Имя Игрушки»

mv "Игрушки для дошколят" "Имя Игрушки"
mv "Игрушки для школьников"/* "Имя Игрушки"
rm -r "Игрушки для школьников"

┌──(kali㉿kali)-[~/lws/attestation2/Имя Игрушки]
└─$ ll
total 0
-rw-r--r-- 1 kali kali 0 Feb 27 08:03  Конструктор.txt
-rw-r--r-- 1 kali kali 0 Feb 27 08:06  Куклы.txt
-rw-r--r-- 1 kali kali 0 Feb 27 08:06  Машинки.txt
-rw-r--r-- 1 kali kali 0 Feb 27 08:06 'Мягкие игрушки.txt'
-rw-r--r-- 1 kali kali 0 Feb 27 08:03 'Настольные игры.txt'
-rw-r--r-- 1 kali kali 0 Feb 27 08:03  Роботы.txt

5)	 Переименовать директорию «Имя Игрушки» в «Игрушки»

mv Имя\ Игрушки 'Игрушки'

6)   Просмотреть содержимое каталога «Игрушки».

┌──(kali㉿kali)-[~/lws/attestation2]
└─$ ll Игрушки
total 0
-rw-r--r-- 1 kali kali 0 Feb 27 08:03  Конструктор.txt
-rw-r--r-- 1 kali kali 0 Feb 27 08:06  Куклы.txt
-rw-r--r-- 1 kali kali 0 Feb 27 08:06  Машинки.txt
-rw-r--r-- 1 kali kali 0 Feb 27 08:06 'Мягкие игрушки.txt'
-rw-r--r-- 1 kali kali 0 Feb 27 08:03 'Настольные игры.txt'
-rw-r--r-- 1 kali kali 0 Feb 27 08:03  Роботы.txt

7)   Установить и удалить snap-пакет. (Любой, какой хотите)

└─$ sudo snap install flappy

8)   Добавить произвольную задачу для выполнения каждые 3 минуты 
	 (Например, запись в текстовый файл чего-то или копирование из каталога А в каталог Б).

sudo crontab -e
